# Quantitative Data Research

## Project Goal
Web-based platform for conducting quantitative data research and analysis.

## Usage
HTML-based application for quantitative research, data analysis, and visualization.

### Running the Application
Open the main HTML file in a web browser:
```bash
open index.html
```

### Features
- Quantitative data analysis
- Research tools
- Data visualization
- Interactive analysis workflows

### Requirements
- Modern web browser
- Local web server (optional, for enhanced functionality)
